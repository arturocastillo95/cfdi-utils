CFDI UTILS
----------


Pequeña librería para trabajar con CFDI. Con esta libreria podemos generar el XML de una factura CFDI 4.0 y sellarlo con el sello digital.

Para ver un ejemplo del uso de la libreria puedes consultar el archivo 'ejemplo_cfdi.py' que se encuentra en el repositorio de GITHUB en la URL: https://github.com/arturocastillo95/cfdi-utils
